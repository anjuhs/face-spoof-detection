# 🧠 Face Spoof Detection using LGS & SVM

A machine learning-based system designed to detect whether a facial image is real or spoof using Local Graph Structure (LGS) features and Support Vector Machine (SVM) classification.

---

## 🚀 Features

* Extract facial features using Local Graph Structure (LGS)
* Train SVM classifier for binary classification
* Predict whether input image is real or spoof

---

## 🧠 Tech Stack

* Python
* MATLAB (original implementation)
* scikit-learn (SVM)
* NumPy

---

## 💡 Methodology

1. Input face image
2. Extract LGS features (local texture relationships)
3. Train SVM classifier using labeled dataset
4. Predict real vs spoof

---

## 📊 Dataset

Originally trained using the **NUAA Photograph Imposter Dataset**.

⚠️ Dataset is not included due to licensing restrictions.
